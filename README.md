# 🚀 Mirodil Mirsadikov - Interactive Resume

Современное интерактивное резюме с темной темой, продвинутыми анимациями и 3D эффектами.

## ✨ Особенности

### 🎨 Визуальные эффекты
- **Темная тема** с градиентными акцентами
- **Анимированные частицы** в фоне
- **3D трансформации** карточек при наведении
- **Голографические эффекты** для заголовков
- **Нейронная сеть** в качестве фонового элемента
- **Matrix rain эффект** для технического стиля
- **Квантовые частицы** с интерактивными связями

### 🎯 Интерактивность
- **Плавная навигация** между секциями
- **Индикатор прогресса** скролла
- **Звуковые эффекты** при клике
- **Cursor trail** эффект
- **Hover анимации** для всех элементов
- **Typewriter эффект** для подзаголовка
- **Text scramble** анимации

### 📱 Адаптивность
- Полностью адаптивный дизайн
- Оптимизация для мобильных устройств
- Поддержка высокого контраста
- Режим уменьшенной анимации
- Печатная версия

### 🎮 Easter Eggs
- **Konami Code** (↑↑↓↓←→←→BA) - активирует Matrix режим
- **Клавиатурная навигация** (j/k, стрелки)
- **Производительность** мониторинг в консоли

## 🛠 Технологии

- **HTML5** - семантическая разметка
- **CSS3** - современные стили и анимации
- **JavaScript ES6+** - интерактивность
- **Canvas API** - продвинутые визуальные эффекты
- **Web Audio API** - звуковые эффекты
- **Intersection Observer** - оптимизированные анимации
- **CSS Custom Properties** - динамические темы

## 🔧 Кастомизация

### Изменение цветов
Отредактируйте CSS переменные в `assets/css/main.css`:
```css
:root {
    --primary-color: #your-color;
    --accent-color: #your-accent;
}
```

### Добавление новых эффектов
Создайте новый класс в `assets/js/effects.js`:
```javascript
class YourEffect {
    constructor() {
        this.init();
    }
    
    init() {
        // Ваш код
    }
}
```

### Настройка анимаций
Измените параметры в `assets/js/main.js`:
```javascript
const animationDuration = 800; // мс
const particleCount = 50;
```
